[1WCCOAR02]
.) Установка с DVD

  Процесс установки программного обеспечения WinCC OA 3.17:

  > Вставьте DVD c WinCC OA в дисковод
  > Смонтируйте CD-ROM, например: 
    
    "mount -t iso9660 /dev/cdrom /cdrom"

  > Перейдите в директорию /cdrom и оттуда в либо linux_rhel_x86_64, либо linux_opensuse_x86_64, в зависимости от используемого дистрибутива Linux.

  > Установите WinCC OA, выполнив следующую команду:
  
    SUSE Enterprise: "zypper install *.rpm"
    RedHat/CentOS: "yum install *.rpm"
 
  Данная команда установит все необходимые пакеты. В качестве альтернативы, можно использовать следующие команды для установки:

    SUSE Enterprise: "zypper install <WinCC_OA_RpmPackageName>.rpm", e.g. "zypper install WinCC_OA_3.17-base-sles-0-0.x86_64.rpm"
    RedHat/CentOS: "yum install <RpmPackageName>.rpm", e.g. "yum install WinCC_OA_3.17-base-rhel-0-0.x86_64.rpm"
  
Ниже перечислены доступные RPM пакеты для WinCC OA:

    WinCC_OA_3.17-base-<distribution>-0-0.x86_64.rpm
      - включает в себя базовую установку плюс некоторые дополнительные функции
    WinCC_OA_3.17-api-<distribution>-0-0.x86_64.rpm 
      - включает WinCC OA API
    WinCC_OA_3.17-applications-<distribution>-0-0.x86_64.rpm
      - включает проекты DemoApplication и GettingStarted
    WinCC_OA_3.17-help-en-<distribution>-0-0.x86_64.rpm
      - включает интерактивную справку на английском языке
    WinCC_OA_3.17-help-de-<distribution>-0-0.x86_64.rpm
      - включает интерактивную справку на немецком языке
    WinCC_OA_3.17-help-ru-<distribution>-0-0.x86_64.rpm
      - включает интерактивную справку на русском языке
    WinCC_OA_3.17-rdb-oracle-libs12-<distribution>-0-0.x86_64.rpm
      - включает Oracle Libraries, необходимые для использования Oracle OCCI Functions (Bulk operations) и функции Reporting вместе с RDB архивированием.
    WinCC_OA_3.17-specialfunctions-<distribution>-0-0.x86_64.rpm 
      - включает WinCC OA API и поддержку боле старых версий
    WinCC_OA_3.17-sqldrivers-<distribution>-0-0.x86_64.rpm
      - включает плагины, необходимые для обращения к различным базам данных, таким как, mySQL, Oracle и др. через CTRL-Ado.
    WinCC_OA_3.17-video-<distribution>-0-0.x86_64.rpm
      - включает Video-AddOn.

II. Требования к системе.

  RedHat Enterprise Linux 8       (64-разрядная)
  CentOS 8                        (64-разрядная)
  SUSE Linux Enterprise Server 15 (64-разрядная)

.) Свободное пространство на жестком диске

При активации Аварийного режима в проекте, потребуется, по крайней мере, 100 Мбайт свободного дискового пространства после установки. В случае если это условие не выполняется, нормальный запуск проекта будет невозможен. Более подробная информация об Аварийном режиме доступна в соответствующей главе интерактивной Справки по WinCC OA.

[2WCCOAR03]  
.) Поддержка JPG

Для обеспечения возможности использования изображений в формате JPG в проекте WinCC OA необходимо установить соответствующую библиотеку (libjpeg62).

[3WCCOAR02]
.) RDB_config.sql

Файл «RDB_config.sql» был переименован в «RDB_config_template.sql», 
поэтому при развертывании обновлений файл «RDB_config.sql» перезаписан не будет. 
При развертывании обновления необходимо убедиться в том, что новые параметры были добавлены в файл «RDB_config_template.sql», 
а существующие параметры были соответствующим образом изменены или удалены. 
Необходимо также внести соответствующие изменения в файл «RDB_config.sql».

В случае первоначальной установки реляционной базы данных необходимо скопировать содержимое файла «RDB_config_template.sql» 
в файл «RDB_config.sql» и внести в этот файл необходимые изменения с учетом особенностей проекта. 
Дополнительная информация доступна в интерактивной Справке по WinCC OA.

[3WCCOAR03]
.) Обновление реляционной базы данных

Существующая реляционная база данных должна быть обновлена при каждой установке новой версии WinCC OA. 
Версия схемы Oracle, используемая в данный момент, отображается в программе просмотра журнала WinCC OA (LogViewer) 
во время загрузки менеджера реляционной баз данных. 

Для обновления схемы Oracle до актуальной версии (8.22) необходимо выполнить 
следующие команды, предварительно (до обновления схемы) внеся необходимые 
изменения в свой файл «RDB_config.sql»:

  cd data\RDBSetup\ora
  unix_upgrade 8.22

[3WCCOAR04]
.) Первоначальная установка
  
В случае первоначальной установки необходимо выполнить следующую команду 
после внесения необходимых изменений в файл «RDB_config.sql»:

  cd data\RDBSetup\ora
  unix_install

[3WCCOAR05]
.) Пакеты RPM

Для организации взаимодействия менеджера РДБ с базой данных Oracle 12 доступны следующие пакеты RPM:

  WinCC_OA_3.17-rdb-oracle-libs12-<distribution>-0-0.x86_64.rpm
  
[3WCCOAR06]
.) Переменные окружения

Переменные окружения должны быть настроены в соответствии с параметрами развертывания систем Oracle и 
WinCC OA. Для получения более подробной информации необходимо обратиться к соответствующему разделу 
интерактивной Справки по WinCC OA: Управление системой > База данных > Архивация РБД > 
Предпосылки и установка > Предпосылки для установки.
